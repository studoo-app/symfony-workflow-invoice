# WorkFlow Invoice

Projet démo du composant workflow Symfony

## Principe du projet

Illustrer le workflow du cycle de vie d'une facture

Editer -> Envoyée -> En attente de reglement -> Réglée / Annulée

Objet facture:
 - createdAt
 - GeneratedAt
 - SendAt
 - PaidAt
 - CancelAt
 - Date
 - Total
 - client
 - Status -> receptacle du workflow
